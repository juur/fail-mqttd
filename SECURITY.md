# Security Policy

If you discover a security vulnerability in **fail-mqttd**, **please do not open a public issue**.

## Reporting a Vulnerability

Please report security issues privately via:

- GitHub: **Security → Report a vulnerability** on this repository  

Include, if possible:

- A description of the issue and its impact
- Steps to reproduce (or proof-of-concept)
- fail-mqttd version/commit and your OS

We will investigate, fix the issue, and publish an advisory. If you’d like credit, please say so in your report.
